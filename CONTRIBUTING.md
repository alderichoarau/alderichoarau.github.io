# Contributing

This repository is the personal portfolio of Aldéric Hoarau. It is published
read-only, as a technical demonstration and showcase.

**External contributions (pull requests, patches) are not accepted.**

- Issues reporting an actual bug (e.g. broken link) are welcome, but no
  guarantee of a fix or a response is given.
- Security vulnerabilities should be reported privately as described in
  [SECURITY.md](./SECURITY.md), not through a public issue.
- Pull requests will be closed without review.
- No forking, reuse, or redistribution of the code is permitted — see the
  [LICENSE](./LICENSE) file.

For any questions, contact alderic.hoarau@gmail.com.
